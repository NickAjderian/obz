welcome First, you need to log in or register Then, if you are a nurse, you need to Join a Ward Or if you are an Administrator you need to Register When you have Registered as and Admin you can then Create a Ward If you are an Administrator or Manager of a Ward then you need to allow Nurses to join the ward Then any user can add Patients Any user can Observe patients during the shift: you'll be notified if an Observation is overdue Any user can run a report of observations during the shift
